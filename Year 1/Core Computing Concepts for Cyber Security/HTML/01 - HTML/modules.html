My Modules

Architecture and Operating Systems

  • This module is all about how the computer processes data, such as the binary system which is the basis of everything the computer does.
  • We then look at the ways that the operating system is able to interact with the computer and the various devices which are connected to it such as a keyboard and mouse.

Core Computing Concepts For Cyber Security

  • In this module we look at some key areas in computing which will help with the course.
  • These four parts are a literature review of AI and the effects that it has on people in their everyday lives.
  • The second is taking a look at web development and making use of HTML and CSS to make interactive websites.
  • The third is some of the core security concepts such as cryptography and access control which is directly linked to the course.
  • The last part is key concepts of usability which will look at how we can make programs that are easy to use for a variety of people including those with disabilities.

Cyber Security And Forensic Essentials

  • This module has two parts, the cyber security part and digital forensics part.
  • For the cyber security part we look at the different threats which computers may be subject to, making use of linux to do this.
  • The computer forensics part involves looking at how we can carry out investigations to devices which might have been used in criminal activity.

Programming

  • This module is all about developing your programming skills in Python. Part of this is making sure that you get into good habits in order to make sure that the code is logical and efficient.
  • We start of by creating simple functions that do basic things such as getting user inputs, doing something with the inputs and then outputting it to the user.
  • Later on in the year we will also be learning the Dart programming language as well.

System Administration and Internetworking

  • This module is all about understanding how networking works within small businesses with a few computers, running both Windows and Linux operating systems.
  • We will be creating multiple virtual machines within themself and networking them together so that they can communicate with each other and have access to the internet.
